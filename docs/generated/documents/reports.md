# Reports Page
================

This section guides you through using the reports page in AIV.

## Loading the Reports Page
---------------------------

To access the reports page, follow these steps:

### Step 1: Navigate to Reports Page
#### What is happening?
The system loads the reports page.
#### Why?
This allows users to view and manage available reports.
#### What is expected?
The reports page should load successfully.

![reports-daily-01-page](../../../screenshots/documents/reports/reports-daily-01-page.png)

### Step 2: Verify Stats Counters Toolbar
#### What is happening?
The system checks if the stats counters toolbar is present.
#### Why?
This ensures that users have access to important report metrics.
#### What is expected?
The stats counters toolbar should be visible.

![reports-daily-02-stats](../../../screenshots/documents/reports/reports-daily-02-stats.png)
