# AIV Reports Section
=====================

The **Reports** section is a central repository for all your reports, providing a comprehensive view of information gathered from diverse sources. Each report includes various visual elements such as charts, graphs, and maps.

## Accessing the Reports Section
-----------------------------

You can access the Reports section by clicking on **Reports**, which opens in the same tab. Alternatively, you can open it in a new tab by clicking the ![Create / Manage Events Icon](../../../../screenshots/documents/reports/create-manage-events-icon.png) icon next to the Reports section.

## Report Overview
-----------------

The report scheduling functionality within the AIV platform allows users to schedule reports efficiently. Reports can be scheduled at various intervals, including:

*   Hourly
*   Daily
*   Weekly
*   Monthly
*   Yearly

Users can generate and export reports in multiple file formats, such as:

### Report File Formats

| Format | Description |
| --- | --- |
| RPT documents | AIV's proprietary report format |
| Excel files (.xlsx) | Microsoft Excel spreadsheet format |
| HTML | Hypertext Markup Language |
| PDF | Portable Document Format |
| PowerPoint presentations (.pptx) | Microsoft PowerPoint presentation format |
| AIV-specific Excel files (.aiv_xlsx) | AIV's proprietary Excel file format |
| QRUN files | AIV's proprietary report format |

## Uploading Reports
------------------

To upload a report, navigate to the Reports section and click on the **Upload** button. You can select the file from your local machine or drag-and-drop it into the file browser.

![Reports page with file browser](../../../../screenshots/documents/reports/reports-daily-01-page.png)

## Viewing Reports
-----------------

Once you have uploaded a report, you can view it by clicking on its name in the grid. The report will be displayed in a new tab, showing various visual elements such as charts, graphs, and maps.

## Scheduling Reports
--------------------

To schedule a report, right-click on the report's name in the grid and select **Schedule** from the context menu. You can choose the scheduling interval (e.g., hourly, daily) and set any additional options.

![Context menu with Schedule option](../../../../screenshots/documents/reports/reports-daily-09-ctx-schedule.png)

## Search Functionality
----------------------

You can search for reports using the search box at the top of the grid. As you type, the grid will filter to show only reports matching your search term.

![Search box with typed input](../../../../screenshots/documents/reports/reports-daily-05-search-typed.png)

### Search Options

*   You can clear the search term by clicking on the **X** button next to the search box.
*   If no matches are found, an empty state or no rows will be displayed in the grid.

![Search with no-match term](../../../../screenshots/documents/reports/reports-daily-07-search-nomatch.png)

## Managing Reports
------------------

You can manage your reports by clicking on the **Manage** button next to each report's name in the grid. This will open a menu with options for:

*   Uploading
*   Viewing
*   Copying
*   Moving
*   Deleting
*   Sharing
*   Hiding Options
*   Auditing
*   Versioning
*   Unarchiving
*   Unpurging

## Saving Reports as Quick Links
-------------------------------

You can save a report as a quick link by clicking on the **Save As Quick Link** button in the grid. This will create a shortcut to the report that you can access from anywhere in the application.

By following these steps and understanding the various features of the AIV Reports section, you should be able to efficiently manage your reports and gain valuable insights from the data.
