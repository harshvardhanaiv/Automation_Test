Introduction
============

Reports serve as data visualizations, providing a comprehensive view of information gathered from diverse sources. Each report includes various visual elements such as charts, graphs, and maps. AIV reports can span one or multiple pages.

The Reports section acts as a central repository for all your reports, enabling efficient management. Users can upload, view, copy, share, delete, download, embed, and save reports as quick links.

Reports are a subsection within the Documents section of the AIV application.

You can access the Reports section by clicking on Reports, which opens in the same tab. Alternatively, you can open it in a new tab by clicking the ![Create / Manage Events Icon](/_astro/newtab.bc2b84d9_ZPwQcr.webp) icon next to the Reports section

#### Report Scheduling Overview

The report scheduling functionality within the AIV platform allows users to schedule reports efficiently. Reports can be scheduled at various intervals, including hourly, daily, weekly, monthly, and yearly.

Users can generate and export reports in multiple file formats, such as:

*   RPT documents
*   Excel files (.xlsx)
*   HTML
*   PDF
*   PowerPoint presentations (.pptx)
*   AIV-specific Excel files (.aiv\_xlsx)
*   QRUN files

For sharing, users can distribute these reports using the built-in email function or share them directly within application with available users.

* * *

#### Primary Purpose

The primary purpose of the **Reports section** is to execute various functionalities associated with reports, including uploading, viewing, managing, scheduling, and sharing reports within the AIV platform.
