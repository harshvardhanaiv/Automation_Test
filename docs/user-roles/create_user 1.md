<h4 className="olive-text"> Follow the steps to create a user: </h4>

##### ✒️ There are two methods through which an admin user can create a user in the AIV application:

1. By clicking on the create button <span style="display: inline-block; vertical-align: middle;">
![Create icon](../../../aiv_img/administration/createuserbutton.png)
</span> located in the bottom-most toolbar.

2. Another option is to right-click in any blank space and then select the 'Create User' button from the context menu, as shown in the following figure:

    ![Image](../../../aiv_img/administration/creat_user_disable1.png)

#####  Now perform the following steps in order to create a new user:

1. Click on Create using one of the methods mentioned above, which will open the following popup.
![Image](../../../aiv_img/administration/create_new_user1.png)


2. Enter the details as shown in the figure above.<br />
3. Let us understand what is the purpose of each field:

* **User Name:** Enter the name you want to give to the user.(*required field)
* **Email Address:** Enter the email address of the user.(*required field)
* **First Name:** Enter the first name of the user.(*required field)
* **Last Name:** Enter the last name of the user.(*required field)
* **Password:**  Password that a user enters while performing login.(*required field)
* **Confirm Password:**  This is to confirm the password; it must match the password field.(*required field)
* **User Type:**  Users can be either internal or external.
* **Backup User:**  This field enables the user to choose a backup user.
* **Status:** This is a toggle value which enables/disables user.(*required field)
* **Admin Active:**  This is a toggle value which gives admin rights to user. If enabled, the user does not need to provide rights to any section individually.
* **Landing Page:**  The first page that will be opened on login.

4. You can determine which sections are visible to the end user by selecting either "No Access," "Only View," or "Full Access" radio buttons. In case of any further help, the user can click on the <span style="display: inline-block; vertical-align: middle;">
  ![Help Icon](../../../aiv_img/common/help.png)
</span> icon.

5. Finally, click on <span style="display: inline-block; vertical-align: middle;">![Submit Icon](../../../aiv_img/administration/aiv_administration_user_create_submit.png)
</span> button to create a new user.  

   ![Image](../../../aiv_img/administration/login_user.png)

:::tip
Login using the following user credentials to review the details of the newly created user.
:::

6. Upon logging in, the user can access all sections of the AIV application for which they have been granted permission.

7. If the user is not satisfied and wants to cancel the creation request,they can do so by clicking on the cancel button next to submit or by clicking on <span style="display: inline-block; vertical-align: middle;">
  ![Cancel Icon](../../../aiv_img/common/cancel.png)
</span> icon next to the <span style="display: inline-block; vertical-align: middle;">
  ![Help Icon](../../../aiv_img/common/help.png)
</span> icon.


8. By following these steps, you can successfully create a new user in the AIV application.
