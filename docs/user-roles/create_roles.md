#### There are two ways in which a user can create a user in the AIV application:

1. By clicking on the create button <span style="display: inline-block; vertical-align: middle;">
  ![CreateRole Button](../../../aiv_img/administration/create_role_button.png)
</span> located at the bottom most toolbar.

2. By right-clicking in any blank space and selecting  'Create Role' from the context menu as follows:

   ![Image](../../../aiv_img/administration/create_role_button3.png)

### Create Role

#### Now perform the following steps in order to create a new user role:

1. Click on Create using one of the methods mentioned above, which will open the following popup.

   ![Image](../../../aiv_img/administration/create_role.png)

2. Enter the details as shown in the figure below.

   ![Image](../../../aiv_img/administration/create_role2.png)

3. Let us understand what is the purpose of each field:
-  **Name:** Enter the name you want to give to the role.(*required field)
-  **Email Address:** Enter the email address of the user role.(*required field)
-  **Description:** Enter any text describing the role of user.(optional field)
-  **Admin Active:** This is a toggle value which gives admin rights to user. If enabled the user does not need to provide rights to any section individually.

4. When a role is not part of the Admin role, the administrator can assign different privileges to the role as per requirements.

5. This can be done using  various options; there are three available for each section: 
-  **No Access:** Users with this role will have no access to the selected sections. 
-  **View:** Users with this role will have only view privileges, that means they cannot edit or create new files in the selected sections.
-  **Full Access:** Users with this role will have all rights, including create, edit, upload and delete for sections with Full Access.

6. In case of any further help, the user can click on the <span style="display: inline-block; vertical-align: middle;">
  ![Help Icon](../../../aiv_img/common/help.png)
</span> icon to access the AIV help documents.

7. Finally, click on <span style="display: inline-block; vertical-align: middle;">
  ![Submit Icon](../../../aiv_img/common/submit_icon.png)
</span> button to create a new user role.

8. If the user is not satisfied and wants to cancel the create request, he can do this by clicking on the cancel button next to submit or by clicking on <span style="display: inline-block; vertical-align: middle;">
  ![Cancel Icon](../../../aiv_img/common/cancel.png)
</span> button next to the <span style="display: inline-block; vertical-align: middle;">
  ![Help Icon](../../../aiv_img/common/help.png)
</span> icon.

9. After clicking the submit button, a success message will appear at the top of the screen.

   ![Image](../../../aiv_img/administration/rolecreatedsuccess.png)
