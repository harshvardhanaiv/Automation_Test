# <h3 className="olive-text"> Follow steps to Assign Roles to Users: </h3> 

- There are two ways in which a user can assign user roles in the AIV application:

1. By clicking on the button located at the bottom most toolbar. ![Image](../../../aiv_img/administration/aiv_administration_users_assignrole_1.png) 

2. Alternatively, users can assign roles by right-clicking on any existing user and selecting the 'Assign Roles by Users' button from the context menu.

   ![Image](../../../aiv_img/administration/editassignuser1.png)

▶️  Now, perform the following steps to assign roles to the user:

1. By using one of the methods mentioned above, the following popup will open.

   ![Image](../../../aiv_img/administration/aiv_administration_assignrole.png)

2. Assign a role to user as per requirement from the list of available roles.

3. In case of any further help, the user can click on the <span style="display: inline-block; vertical-align: middle;">
  ![Help Icon](../../../aiv_img/common/help.png)
</span>  icon to access the AIV help documents.

4. Finally, click on <span style="display: inline-block; vertical-align: middle;">
  ![Submmit Icon](../../../aiv_img/administration/aiv_administration_department_submit_button.png)
</span>  button to assign the role to the user. 

5. After assigning the role, log in with the user credentials provided below to check the details. For example, if the user is assigned the Administrator role, they will have access to all sections of the AIV application.

   ![Image](../../../aiv_img/administration/login_user.png)

6. If the user is not satisfied and wants to cancel the create request, he can do this by clicking on the cancel button next to submit or by clicking on <span style="display: inline-block; vertical-align: middle;">
  ![Cancel Icon](../../../aiv_img/common/cancel.png)
</span> button next to the <span style="display: inline-block; vertical-align: middle;">
  ![Help Icon](../../../aiv_img/common/help.png)
</span> icon.

7. In this manner, you can assign a role to the user in the AIV application.
