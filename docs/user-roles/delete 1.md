# <h3 className="olive-text"> Follow steps to Delete User : </h3>

#### There are two ways in which a user can be deleted:

1. By clicking on the  button located at the bottom most toolbar.![Image](../../../aiv_img/administration/aiv_administration_user_delete.png)<br />
2. By right clicking any existing user and select Delete as follows.

![Image](../../../aiv_img/administration/deleterole.png)

#### Now perform the following steps in order to delete a user:

1. Using any of the above mentioned methods click on delete which will open the following dialog box.

![Image](../../../aiv_img/administration/aiv_administration_user_delete_conformation.png)

2. In case of any further help, the user can click on the ![Image](../../../aiv_img/common/help.png) icon to access the AIV help documents.

3. Finally, click on button to delete the user.![Image](../../../aiv_img/administration/aiv_administration_department_delete_button.png) 

4. If the user is not satisfied and wants to cancel the delete request, he can do this by clicking on the cancel button next to submit or by clicking on button next to the![Image](../../../aiv_img/common/cancel.png)  ![Image](../../../aiv_img/common/help.png)

5. By following above steps, you can delete a user in the AIV application.
