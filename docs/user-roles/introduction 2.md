1. Roles section in the AIV application enables the end-users to create new user roles.

2. However, end-users can only create user roles up to the limit of their license. If they attempt to create more roles, an error will be generated. 

:::note
Check License details under Administration.
:::

3. Roles tab can be accessed from the main menu. Roles and other sections under the Administration tab are only visible if the appropriate privileges have been assigned to the logged in user, Please contact your administrator if you facing difficulties accessing the sections under the Administration tab.  

### ✒️  Role section contains following features: 

1️⃣ **Create Role:** enables admin user to create role as per requirements.

2️⃣ **Edit User Role:** enables admin user to do modification in existing user's role.

4️⃣ **Delete Role:** allows Admin user to delete the role.
