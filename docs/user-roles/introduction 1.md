<h4 className="olive-text"> The Administration tab contains User section as follows:</h4>

![users](../../../aiv_img/administration/users_intro1.png) 

* The Users section in the AIV application enables end-users to create new users and modify existing ones.

* However, end-users can only create as many users as they are licensed for, and attempting to exceed the licensed limit will result in an error. (Refer to the License details under Administration for more information on this).

* User tab can be accessed from the main menu under the Administration section. Sections such as Users and others under the Administration tab are only visible if the logged-in user has been assigned the appropriate privileges 🔒. Please contact your administrator if you encounter difficulties accessing the sections under the Administration tab.

#### ✒️  Users section contains the following features: 

- **Create** User enables the admin user to create a user according to the requirements.

- **Edit User** enables admin user to do modification in existing user.

- **Edit User Role**  is used to assign a role to the user from the available list.

- **Delete** allows the admin user to remove a user.

- **Assign Role** allows the assignment of specific roles to users from the available list, ensuring a seamless and efficient workflow.
