* There are two ways in which a user role can be deleted: 

**(A)** By clicking on the ![Image](../../../aiv_img/common/delete.png) button located at the bottom most toolbar.<br />
**(B)** By right clicking any existing user role and select Delete as follows: 

![Image](../../../aiv_img/administration/deleterolecontext.png)

### Delete role

* Now perform the following steps in order to delete a user: 

1. Using any of the above-mentioned methods, click on Delete, which will open the following dialog box.

![Image](../../../aiv_img/administration/delete_role.png)

2. In case of any further help, the user can click on the ![Image](../../../aiv_img/common/help.png) icon to access the AIV help documents.

3. Finally, click on ![Image](../../../aiv_img/common/delete.png) button to delete the user role.

4. If the user is not satisfied and wants to cancel the delete request, user can do this by clicking on the cancel button next to submit or by clicking on ![Image](../../../aiv_img/common/cancel.png) button next to the ![Image](../../../aiv_img/common/help.png) icon.

5. As user click on the Delete button, the alert appear on the top of the screen.

![Image](../../../aiv_img/common/deletesuccessalert.png)
